# 🍱 What2Eat (今天吃啥)

> 解决全人类每天都会面临的世纪难题：“今天吃什么？”

What2Eat 是一个轻量级的全栈 Web 应用，旨在帮助有选择困难症的用户快速决定就餐选择。通过简单的交互，告别点餐前的纠结。

## ✨ 核心功能

**目前已实现：**
- [x] **菜品管理**：支持自定义添加各种菜品到你的专属“菜篮子”。
- [x] **随机盲盒**：一键随机抽取今天的就餐决定，公平公正，专治选择困难。

**🚀 未来规划 (To-Do)：**
- [ ] **标签分类**：支持按类别（如面食、米饭、快餐、日料）进行筛选后再抽取。
- [ ] **权重系统**：为偏爱的菜品增加中选概率，或设置近期吃过菜品的“冷却时间”。
- [ ] **社交分享**：支持一键导出/导入个人专属菜单。
